## 准备
### 环境
- Node 14.18+
- pnpm

::: details 选择 pnpm 的原因

npm 会缓存此前下载的版本，因此可能

:::

### 安装
::: code-group
```sh [创建项目]
pnpm create valaxy
```

```sh [创建项目]
cd valaxy-blog
pnpm i
pnpm dev
```
:::
